Đăng nhập

Mật khẩu chỉ dùng để đổi lấy phiên đăng nhập trên thiết bị này — app không lưu token GitHub thật, token thật nằm ở server trung gian (Cloudflare Worker). Mỗi thiết bị chỉ cần đăng nhập 1 lần, phiên có hiệu lực 90 ngày.
