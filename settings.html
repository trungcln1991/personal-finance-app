Kết nối GitHub

Cách tạo token: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Repository access: chọn Only select repositories → personal-finance-data. Permissions: Contents → Read and write. Không cấp thêm quyền nào khác. Token chỉ lưu trong trình duyệt của bạn (localStorage), không gửi đi đâu khác ngoài GitHub API.
